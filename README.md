### 📝 진행 방식
- 일시: 매주 목요일 오후 10시
- 장소: 디스코드 자율 스터디룸
- 매주 2명의 발표자가 주제 하나씩 발표한다.
- 나머지 인원도 동일한 2개 주제를 각자 정리해오고, 발표 후 질문한다.
- 정리한 내용은 스터디 시간 전 깃허브 레포지토리에 커밋한다.
